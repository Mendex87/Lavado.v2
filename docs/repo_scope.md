# Repo Scope

## Entra al repo
- backend real del proyecto
- schemas SQL
- documentación técnica y funcional del proyecto
- preview/prototipos que ayuden a validar UI/flujo
- assets del producto

## Queda fuera del repo
- memoria personal del asistente
- bootstrap y archivos internos del workspace OpenClaw
- secretos y configuraciones locales
- caches y runtime local

## Motivo
Mantener el repo portable, limpio y compartible entre VPS, PC del taller y cualquier otra máquina de desarrollo.
